# [기획서] 온프레미스 엔지니어링 문서 무결성 검증 애플리케이션 (LEDIC)

## 1. 프로젝트 개요
* **프로젝트명:** LEDIC (Local Engineering Document Integrity Checker)
* **추진 배경:**
    * 플랜트 설계 과정에서 P&ID, Datasheet, Equipment List 간의 기술 데이터 불일치는 심각한 재작업 및 안전 문제를 초래함.
    * 사내 보안 및 망분리 규정으로 인해 웹 브라우저 기반 업로드나 외부 API(OpenAI 등) 사용이 원천적으로 불가함.
* **목적:** 인터넷 연결 없이 사내 PC에서 독립 구동되는 데스크톱 앱(.exe)을 통해 문서 간 데이터 통일성을 자동으로 교차 검증함.
* **기대 효과:** 단순 대조 업무의 자동화(에러 감소), 완벽한 데이터 보안 유지, 주니어 엔지니어 생산성 향상.

## 2. 기술 스택 및 채택 사유
본 시스템은 모든 데이터를 로컬 메모리 내에서만 처리하며, 외부 통신이 차단된 **Standalone** 구조를 가집니다.

| 구분 | 기술/도구 | 채택 사유 |
| :--- | :--- | :--- |
| **GUI 프레임워크** | **PySide6** | 공식 Qt 라이브러리. **LGPL 라이선스**로 사내 비공개 프로그램 개발 및 무료 배포 가능. |
| **영역 탐지 (Vision)** | **YOLOv11** | 도면 내 Spec Box, Table 등 특정 영역을 고속으로 인식하여 이미지 조각(Crop) 생성. |
| **텍스트 추출** | **PaddleOCR** | 로컬 구동 지원. 도면 특유의 텍스트 배치 및 폰트에 강점이 있는 오픈소스 OCR. |
| **데이터 정형화** | **sLLM (Llama-3 8B)** | 추출된 비정형 텍스트를 JSON 형태의 데이터(Key-Value)로 변환. (Ollama 활용) |
| **배포 패키징** | **PyInstaller** | 파이썬 환경과 AI 모델을 하나로 묶어 설치가 필요 없는 **.exe** 파일로 제작. |

## 3. 핵심 기능 명세

### 3.1. 통합 데이터 검증 및 시각화
* **전체 리스트 표기:** 마스터 EQ List를 기준으로 P&ID와 Datasheet에서 추출된 모든 데이터를 하나의 테이블에 표시.
* **불일치 음영 처리(Shading):** 일치하는 데이터는 일반 표기하고, **값이 서로 다른(Mismatch) 셀에만 특정 배경색으로 음영 처리**하여 검토 편의성 극대화.

### 3.2. 시각적 비교 뷰어 (Side-by-Side)
* 특정 기기의 데이터를 클릭 시, 하단 화면에 **[P&ID 캡쳐본]**과 **[Datasheet 캡쳐본]**을 나란히 배치.
* YOLOv11이 탐지한 좌표를 기반으로 원본 문서에서 즉시 해당 영역을 크롭하여 보여줌으로써 엔지니어의 육안 검증 지원.

### 3.3. 엑셀 증빙 보고서 생성
* 검증 완료 후 결과를 Excel로 내보낼 때, 텍스트 데이터뿐만 아니라 **불일치 항목의 캡쳐 이미지 자체를 엑셀 셀 내에 삽입**하여 직관적인 펀치 리스트(Punch List) 생성.

## 4. AI 모델 학습 및 접목 방안 (YOLOv11)
사내 표준 및 벤더 포맷에 최적화하기 위해 **전이 학습(Transfer Learning)**을 적용합니다.

1.  **데이터셋 구축:** 사내 P&ID 및 Datasheet 약 100장을 이미지로 변환.
2.  **라벨링:** 주니어 엔지니어가 `Spec Box`, `Tag No`, `Table` 영역을 마우스 드래그로 라벨링.
3.  **학습:** GPU 탑재 워크스테이션에서 YOLOv11 베이스 모델을 추가 학습시켜 사내용 가중치 파일(`best.pt`) 확보.
4.  **접목:** PySide6 앱 실행 시 해당 모델을 로드하여 PDF 업로드 즉시 영역 탐지 수행.

## 5. 시스템 요구사항
* **OS:** Windows 10/11 (64-bit)
* **Hardware:** RAM 16GB 이상, NVIDIA GPU(VRAM 8GB+) 권장 (GPU 부재 시 CPU 추론 모드로 동작)
* **보안 환경:** 완전 폐쇄망 및 망분리 환경 지원 (인터넷 불필요)

## 6. 개발 로드맵
* **1주차:** 데이터 수집 및 라벨링 (영엔지니어 협업)
* **2주차:** YOLOv11 학습 및 sLLM 프롬프트 최적화
* **3주차:** PySide6 기반 UI 개발 및 AI 엔진 연동
* **4주차:** 통합 테스트 및 .exe 배포 파일 제작
